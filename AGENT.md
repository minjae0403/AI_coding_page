# 프로젝트 가이드라인 (AGENT.md)

이 문서는 본 프로젝트의 개발 및 유지보수를 위한 기술적 지침을 담고 있습니다.

## 기술 스택
- **프레임워크:** React, Vite
- **언어:** TypeScript
- **스타일:** Tailwind CSS
- **UI 컴포넌트:** shadcn/ui

## 핵심 개발 원칙
1. **컴포넌트 재사용:** UI 컴포넌트는 `src/app/components/ui`에 정의된 shadcn/ui 컴포넌트를 우선적으로 활용합니다.
2. **아키텍처 패턴:** 상태 관리는 가급적 상위 컴포넌트(`App.tsx`)에서 제어하며, 하위 컴포넌트로 props를 통해 전달하는 방식을 선호합니다.
3. **안정성:** 별점 렌더링 및 수치 계산 시, `Math.min`/`Math.max` 등을 사용하여 예외 상황(범위 초과 등)을 방지합니다.
4. **일관성:** 기능 추가 시 `README.md`를 최신 상태로 유지합니다.

## 개발 명령어
- **개발 서버 실행:** `npm run dev`
- **빌드:** `npm run build`
